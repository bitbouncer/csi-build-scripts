csi-build-scripts
=================
common buildscripts for bitbouncer repos download and build all required libs

raspberry crosscompile on ubuntu 32 bit host

install tools
```
wget https://github.com/bitbouncer/csi-build-scripts/raw/master/raspberry_setup_ia32.sh && sudo bash ./raspberry_setup_ia32.sh
```

download source and rebuild
```
wget https://github.com/bitbouncer/csi-build-scripts/raw/master/raspberry_rebuild_ia32.sh && bash ./raspberry_rebuild_ia32.sh
```
